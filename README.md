# Snakes And Ladders

Webxdc port of https://github.com/Hope41/snakes-and-ladders by Joachim Ford

> Help Drillo on his mission to destroy the X-Ray Orb by avoiding the dangerous side effects and gaining power-ups and upgrades along the way.

Snakes And Ladders is a fast-paced JavaScript platform game. You can play it online [here](https://joachimford.uk/snakes-and-ladders/), but please give it a star if you enjoyed it!
